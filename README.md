# Gestor de Tareas

- aplicación que permite elegir entre una serie de opciones, las cuales se detallan a continuación:

	1. Capaz de crear tareas, cun un nombre, una descripción y un identificador.
	2. Mostrar las tareas creadas.
	3. Listar las tareas.
	3. Borrar las tareas.
